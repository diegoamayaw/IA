1) Se debe tener el compilador de Common Lisp "SBCL" y Processing para poder ejecutar el programa.

2) Se debe tener en un mismo directorio todos los archivos decargados en la forma en la que están jerárquicamente (el archivo .lisp y la carpeta GUI con los archivos PDE y el shell).

3)En el archivo prueba.sh se debe editar en donde se carga el archivo connect4.lisp poniendo la ruta exacta en donde se encuentra la carpeta descargada.

4)Se debe abrir con Processing el archivo Tablero.pde y ejecutarlo para poder probar el programa.

5)Se debe seleccionar la dificultad deseada y comenzar a jugar siendo el usuario el primero en hacerlo.
