# Artificial-Intelligence
Projects from college Artificial Intelligence subjects 

Para ejecutar el programa se requiere de una versión igual o superior a 3.6.*
Se recomienda usar spyder de anaconda para una mejor visualización de los grupos y la tabla de asignaciones de una forma más amigable.

Nota: Las restricciones referentes a grupos y las relaciones se tienen que mandar como string desde la entrada i.e. "relacion_a_ingresar" o "grupo_a_restringir"
